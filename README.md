## 作业一  flappy_cell.html cell.js
模仿FlappyBird游戏设计出的简易版FlappyCell游戏，基于HTML和JavaScript。

上方向键控制移动，方块出边界、触碰障碍都会死亡，系统提示并显示得分后可自动重新加载，得分为当前所有障碍的数目。

函数
generateCell(); // 生成方块

generateObstacles(); // 生成障碍

isDeath(); // 判断死亡

attBottom(a, p); // 触底部障碍

attTop(a, p); // 触顶部障碍

Rand(min, max); // 生成min至max的随机数

loop(); // 程序循环函数
